# 3d-storage

Отображение движения товаров внутри склада с ячейками для хранения товаров на тепловой карте

Перед запуском должен быть установлен Node.js.

Для запуска выполнить команды:
```
git clone https://github.com/Sumeruk/3dMag.git

npm run dev
```
